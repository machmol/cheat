Headings

Tags

Tag Name Description
<h1> heading 1 This is the main heading
<h2> heading 2 This is the first subheading
<h3> heading 3 This is the third heading
<h4> heading 4 This is the fourth heading
<h5> heading 5 This heading is quite small
<h6> heading 6 This is the smallest heading

Tables

Tags

Tag Name Description
<table> Table This tag wraps around the table content
<thead> Table Head This tag wraps around the first table row and defines the tables head
<tbody> Table Body This Tag wraps around the tables Main Content
<tfooter> table footer This tag wraps around the last row of the table and defines its footer
<th> table heading This tag defines the first ("heading") cell of a column or row
<td> table data this tag defines a cell within the table

Images

Tags

Tag Name Description
<img> Image This Tag includes an Image to your page

Attributes

Attribute Name Description
src Source Contains the URL to the image
alt alternative Contains an alternative text for if the image cannot be rendered.

Videos

Tags

Tag Name Description
<video> Video This Tag includes a Video to your page

Attributes

Attribute Name Description
src Source Contains the URL to the Video
controls Controls Ads the browser implemented control elements to the video
